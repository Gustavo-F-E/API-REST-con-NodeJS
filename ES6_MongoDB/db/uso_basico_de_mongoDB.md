# Manual de uso muy basico de mongo DB

## Usar una base de datos

```
use veoveo
```

## Crear colecciones dentro de esa base de datos

```
db.createCollection("consultas")
db.createCollection("peliculas")
db.createCollection("series")
db.createCollection("usuarios")
```

## Insertar varios registros dentro de una coleccion

```
db.consultass.insertMany([
.... objetos del json ....
])
```

Ejemplo:
```
db.consultass.insertMany([
  {
    _id: "1",
    nombre_y_apellido: "Juan Pérez",
    tipo_consulta: "Soporte Técnico",
    URL_captura_problema: "http://example.com/captura1",
    descripcion_problema: "Problema con la conexión a internet"
  }
])
```

## Importacion de archivos JSON (o CSV)  
También es importante que que puede importar a MongoDB directamente un archivo JSON.  
Es importante recalcar que lo que queremos que sea nuestro ID, se debe modificar 
en el JSON (colocando en la clave "_id") para que MongoDB no nos genere el ID aleatorio
que trae por defecto.
